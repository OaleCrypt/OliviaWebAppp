How to Access Each URL:
To access any of these URLs, you can navigate directly in your browser to the corresponding path. Assuming your server is running on http://127.0.0.1:8000/, you can append the URL paths to this base URL.

For example:

To view the alerts list, go to http://127.0.0.1:8000/soc/alerts/.
To view the Shodan API data, go to http://127.0.0.1:8000/integrations/shodan/.
